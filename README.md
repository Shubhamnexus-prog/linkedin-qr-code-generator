# 🔗 LinkedIn QR Code Generator

A professional Python-based application that generates high-quality QR codes linking directly to a LinkedIn profile. This project is designed to help users easily share their professional identity through a simple scan.

---

## 🚀 Features

- ✅ Generate QR code for LinkedIn profile  
- 🎨 Customizable design (colors, size)  
- 🖼️ Logo embedding (Pro version)  
- 📥 High-quality image output (PNG)  
- ⚡ Fast and lightweight  

---

## 🛠️ Technologies Used

- Python 🐍  
- qrcode library  
- Pillow (PIL)
